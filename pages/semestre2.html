Algoritmos e Programação de Computadores I

COM110 80h
Saiba Mais

Objetivos

Compreender conceitos básicos de programação e descrever algoritmos para resolver problemas utilizando a linguagem de programação Python.

Ementa

Algoritmos: caracterização, notação, estruturas básicas. Conceitos de linguagens algorítmicas: expressões; comandos sequenciais, seletivos e repetitivos; variáveis; listas; funções; Desenvolvimento e depuração de programas utilizando linguagem Python.

Bibliografia Base

  • PERKOVIC, Ljubomir. Introdução à Computação Usando Python. LTC, 2016.
  • MILLER, Brad; RANUM, David. Como Pensar como um Cientista da Computação. 2019.

Cálculo I

MCA001 80h
Saiba Mais

Objetivos

Desenvolver os conceitos e apontar aplicações do Cálculo Diferencial e Integral de funções de uma variável real.

Ementa

Funções. Limite e Continuidade. Derivada: Definição, Regras de Derivação, Aplicações. Integral: Definição, Técnicas de Integração, Teorema Fundamental do Cálculo.

Bibliografia Base

  • GUIDORIZZI, H. L. Um curso de cálculo: volume 1. Rio de Janeiro: LTC, 2013.
  • STEWART, J. Cálculo: volume 1. São Paulo: Cengage Learning, 2013.

Introdução a Conceitos de Computação

COM140 40h
Saiba Mais

Objetivos

Fornecer noções básicas sobre o funcionamento de dispositivos computacionais e sobre álgebra booleana.

Ementa

Breve história da computação. Sistemas de Numeração. Operações lógicas e tabela da verdade. Representação de números. Tipos de Dispositivos, Arquivos, Redes e Sistemas Operacionais.

Bibliografia Base

  • Brookshear, J. Glenn. Ciência da Computação: Uma Visão Abrangente. 2013. Bookman.
  • CARVALHO, André C. P. L. F. de; LORENA, Ana Carolina. Introdução à Computação. 2016. LTC.

Algoritmos e Programação de Computadores II

COM120 80h
Saiba Mais

Objetivos

Reforçar a prática de programação em Python, composição de programas com múltiplos arquivos, uso de bibliotecas, APIs e GUIs. Noções de programação orientada a objetos.

Ementa

Conceito e uso de pilhas, filas, listas, árvores. Recursão. Algoritmos de Ordenação e Busca. Módulos e noções de objetos. Uso de APIs básicas da WEB. Criação de interfaces gráficas simples. Controle de Versão (git).

Bibliografia Base

  • PERKOVIC, Ljubomir. Introdução à Computação Usando Python. LTC, 2016.
  • MILLER, Brad; RANUM, David. Como Pensar como um Cientista da Computação. 2019.

Fundamentos de Internet e Web

COM130 40h
Saiba Mais

Objetivos

Apresentar noções de conhecimentos sobre a estrutura de documentos na WEB.

Ementa

Hipertexto; origens de XML e HTML; Estrutura do conteúdo versus aparência do documento; tags HTML básicas e extensíveis; Atributos básicos de CSS, seletores CSS; formulários HTML.

Bibliografia Base

  • MILETTO, Evandro Manara; BERTAGNOLLI, Silvia de Castro. Desenvolvimento de Software II. 2014. Bookman.
  • TERUEL, Evandro Carlos. HTML 5 - Guia Prático. 2014. Érica.

Fundamentos Matemáticos para Computação

COM150 80h
Saiba Mais

Objetivos

Desenvolver a base matemática necessária ao projeto de algoritmos e noções de complexidade.

Ementa

Conceitos básicos de matemática discreta e de lógica para computação. Técnicas de provas, indução matemática. Relações e conceitos de teoria de grafos. Modelagem de problemas usando grafos.

Bibliografia Base

  • GERSTING, Judith L. Fundamentos Matemáticos para a Ciência da Computação. LTC, 2016.
