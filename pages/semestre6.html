Infraestrutura para Sistemas de Software

COM310 80h
Saiba Mais

Objetivos

Compreender e aplicar conceitos de comunicação através de redes de computadores e forma de organização de sistemas em nuvem.

Ementa

Redes de computadores e a internet. Noções de redes TCP/IP. Máquinas virtuais. Computação em nuvem. Contêineres e desenvolvimento usando contêineres. Noções de uso do docker e do openshift.

Bibliografia Base

  • TANENBAUM, A. S.; J. WETHERALL, D. J. Redes de Computadores. Pearson, 5ª Ed., 2011.
  • Vitalino, Jeferson Fernando Noronha; Castro, Marcus André Nunes. Descomplicando o Docker. Brassport, 2018.

Plataforma de Ingestão e Análise de Dados

COM420 80h
Saiba Mais

Objetivos

Desenvolver conceitos e práticas para coleta e análise de dados em sistemas de internet das coisas.

Ementa

Nuvens computacionais: características essenciais, modelos de serviço. Integração de dispositivos IoT a nuvens computacionais. Soluções de armazenamento de dados. Plataformas para IoT (AWS, Azure, IBM Bluemix).

Bibliografia Base

  • TANENBAUM, A. S.; J. WETHERALL, D. J. Redes de Computadores. Pearson, 5ª Ed., 2011.
  • Stallings, William. Arquitetura e organização de computadores. Pearson, 2017.

Desenvolvimento Web

COM320 80h
Saiba Mais

Objetivos

Desenvolver sistemas de gestão de informação e de conteúdo através de aplicações Web.

Ementa

Noções de escalabilidade e concorrência. Arquiteturas Web (3 camadas, MVC). Tecnologias de Interface Gráfica com Usuário baseada em Javascript (AngularJS, Vue.js, React). Frameworks back-end (Django). Noções de segurança da informação.

Bibliografia Base

  • MILETTO, Evandro Manara; BERTAGNOLLI, Silvia de Castro. Desenvolvimento de Software II. Bookman, 2014.
  • Santos, Valquiria. Projeto de interface com o usuário. Pearson, 2017.

Interface Humano-Computador

COM430 80h
Saiba Mais

Objetivos

Desenvolvimento de interfaces de usuário com ênfase em WEB com noções de acessibilidade.

Ementa

Histórico e evolução de IHC. Métodos e técnicas de design. Experiência de usuário, arquitetura da informação e acessibilidade. Páginas responsivas. Manipulação do DOM com JavaScript. REST e AJAX.

Bibliografia Base

  • BENYON, D. Interação Humano-computador. Pearson, 2ª Ed., 2011.
  • Rocha, H.V.; Baranauskas, M.C.C. Design e Avaliação de Interfaces Humano-Computador. NIED, 2003.
